pri.py - Main PFR Response Interface update execution

htm.py - Functions for string Manipulation to develop HTML code for pages 

index.py - Run metrics, build index html page

metrics.py - Functions to do analysis and build metric graphics 
    for periodic summaries

event.py - Build event file structure, run event analysis, update 
    periodic logs, build trend images

filestr.py - Functions for modifying the file structure (e.g., creating new event directories, building log files to be written to, etc.)
